## 📱 App Overview

**Telegram** is a cloud-based messaging platform known for its speed, security, and features like channels, bots, and large group chats.

:root {
  --telegram-primary: #4A90E2;         /* Blue - Trust */

  --telegram-received: #F0F2F5;         /* Light Gray - Received */

}
