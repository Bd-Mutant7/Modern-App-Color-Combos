## Best Practices for Medical Apps

### Color Guidelines
- Use calming colors (teals, blues, soft greens)
- Ensure high contrast for accessibility (minimum 4.5:1)
- Avoid red-green combinations for color-blind users
- Use coral or orange for alerts/urgency

### Typography
- Minimum font size: 16px for body text
- Line height: 1.5 for readability
- Use sans-serif fonts for better screen readability

### UI Components
- Buttons: Minimum 44x44px touch target
- Forms: Clear labels and error states
- Navigation: Consistent placement
- Cards: Use shadows for depth perception

### Accessibility
- Support screen readers
- Provide alt text for images
- Ensure keyboard navigation
- Test with grayscale mode

### Emergency Features
- Emergency contact button should be red (#FF6B6B)
- Crisis resources must be easily accessible
- Clear visual distinction for urgent information
