## Key Principles

### 1. Clarity First
- Use clear, simple language (avoid medical jargon)
- Provide definitions for medical terms
- Use icons with text labels
- Maintain consistent terminology

### 2. Error Prevention
- Confirmation dialogs for critical actions
- Undo options where possible
- Clear error messages with solutions
- Prevent accidental data deletion

### 3. Data Visualization
- Use green (#00A86B) for normal results
- Use coral (#FF6B6B) for abnormal results
- Include trend indicators (graphs)
- Show historical data for context

### 4. Trust Indicators
- Display security badges
- Show HIPAA compliance
- Include privacy policy links
- Verify healthcare provider credentials

### 5. Emergency Protocols
- Prominent emergency contact
- One-tap call to emergency services
- Crisis resource links
- Clear instructions for urgent situations
